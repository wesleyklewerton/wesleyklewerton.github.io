MECBA-Clu-implementation

Diretórios:
 - /MECBA-Clu/DIST -> Possuir o arquivo .jar da ferramenta
 - /MECBA-Clu/PROBLEMAS -> Contém os arquivos com os dados dos sistemas utilizados para a implemetação
 - /MECBA-Clu/RESULTADOS -> Recebe os resultados das execuções
 - /MECBA-Clu/SRC -> Código fonte da ferramenta
 - /MECBA-Clu/src/jmetal/experiments/ -> Diretório com as classes executáveis 

Arquivos:
 - /MECBA-Clu/command_to_run.sh -> Comandos para executar a ferramenta e gerar um arquivo TXT com o runtime
